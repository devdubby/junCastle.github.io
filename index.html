영화, 드라마 리뷰 정보 웹 프로젝트

프로젝트 목적

기존에 영화, 드라마 리스트만을 불러오던 클론코딩 인터넷 강의의 결과물에서 영화 예고편, 시리즈, 리뷰, 좋아요 등 더 다양한 정보들을 보여주기 위해 개발한 영화, 드라마 정보 웹 페이지

프로젝트 설명

사용하고 있던 공용 영화 API에서 국문으로 된 영화 리뷰 데이터들을 얻을 수 없었습니다. 이를 해결하고 웹의 기능 확장성을 고려하여 Jwt Token을 활용한 User 인증 로직을 개발했고, 영화 리뷰 정보들을 볼 수 있도록 리뷰, 좋아요 기능의 CRUD API를 개발했습니다. 기본 UI는 노마드코더의 인터넷 강의를 들으면서 클론코딩으로 진행했고, 이후 다시 처음부터 개발하며 Style을 설계했습니다. 자유도가 높은 Mobx보다는 상태관리의 교과서적인 이론을 습득하기 위해 Redux를 이용했습니다.

스터디 모집 웹 프로젝트

프로젝트 목적

ReactJS를 활용한 프론트엔드 개발 학습을 위해 진행한 팀 프로젝트

프로젝트 설명

스터디를 찾거나 모집할 수 있는 웹 페이지 입니다. 스터디 모집 관련 플랫폼이나 커뮤니티들은 많았지만 대부분 딱딱한 화면 디자인과 스터디의 위치 정보를 한눈에 보기 어렵다는 문제점이 있었습니다. 저희는 이를 개선하고자 React 기반의 UI framework를 활용하여 더욱 깔끔하고 이용자 친화적인 화면을 구성하려 했습니다. Naver Map API를 활용하여 이용자의 위치 기반으로 스터디를 찾거나 모집할 수 있도록 하려 했습니다. 또한, multer모듈을 활용해 스터디의 커버 이미지를 올릴 수 있도록 하여 스터디의 신빙성을 확보하고 Socket.io를 활용해 쪽지를 주고받을 수 있도록 하여 이용자들간의 교류가 가능하도록 했습니다. 스터디 정보는 한번 올려놓으면 특별한 작업이 많이 필요하지 않기때문에 상태관리 라이브러리를 이용하지 않고 Context API를 이용했습니다.
