스터디 모집 웹 프로젝트

프로젝트 목적

ReactJS를 활용한 프론트엔드 개발 학습을 위해 진행한 프로젝트

프로젝트 설명

스터디를 찾거나 모집할 수 있는 웹 페이지 입니다. 스터디 모집 관련 플랫폼이나 커뮤니티들은 많았지만 대부분 딱딱한 화면 디자인과 스터디의 위치 정보를 한눈에 보기 어렵다는 문제점이 있었습니다. 저희는 이를 개선하고자 ReactJS 기반의 UI framework를 활용하여 더욱 깔끔하고 이용자 친화적인 화면을 구성하려 했습니다. Naver Map API를 활용하여 이용자의 위치 기반으로 스터디를 찾거나 모집할 수 있도록 하려 했습니다. 또한, multer모듈을 활용해 스터디의 커버 이미지를 올릴 수 있도록 하여 스터디의 신빙성을 확보하고 Socket.io를 활용해 쪽지를 주고받을 수 있도록 하여 이용자들간의 교류가 가능하도록 했습니다.

웹 채팅 프로젝트

프로젝트 목적
